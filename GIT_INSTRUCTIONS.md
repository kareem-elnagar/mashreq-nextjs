# GitHub Repository Setup Instructions

Since Git is not currently installed on this environment, you will need to follow these steps manually once you have installed Git (from [git-scm.com](https://git-scm.com/)).

## 1. Initialize Local Repository
Open your terminal in the `mashreq-nextjs` folder and run:
```powershell
git init
git add .
git commit -m "Initial commit: Mashreq Solar Website (Next.js)"
```

## 2. Connect to GitHub
1. Create a new, empty repository on [GitHub](https://github.com/new).
2. Run the following commands (replace `YOUR_USERNAME` with your GitHub username):
```powershell
git remote add origin https://github.com/YOUR_USERNAME/mashreq-solar.git
git branch -M main
git push -u origin main
```

## 3. Deployment (Optional)
This project is optimized for deployment on **Vercel**. 
- Simply push your code to GitHub.
- Log in to [Vercel](https://vercel.com/) and import the repository.
- Vercel will automatically detect Next.js and deploy your site.
