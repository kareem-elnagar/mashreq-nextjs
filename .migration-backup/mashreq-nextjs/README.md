# Mashreq Solar — Engineering for Real Operation

This is the official web platform for **Mashreq Solar**, an engineering company specializing in high-performance solar systems designed for real-world field conditions (heat, dust, and load variations).

## 🚀 Technical Stack
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🎨 Design Philosophy: "Smart & Easy"
The site uses a premium, engineering-focused aesthetic:
- **Contrast**: High-legibility text on tinted backgrounds to reduce glare.
- **Hierarchy**: Strong visual flow from technical proof to detailed explanation.
- **Aesthetics**: Glassmorphism, smooth gradients, and extreme rounding (`3rem`) for a modern, approachable feel.

## 🛠 Development

### Setup
```bash
npm install
```

### Run Locally
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

## 📂 Project Structure
- `/src/app`: Application routes and pages.
- `/src/components`: Reusable UI components (Navbar, Sections).
- `/src/lib/data.ts`: Central source of truth for technical assessments and project data.
